GIT CHEAT SHEET

1. Konfiguracija
----------------
git config --global user.name "Ime Prezime"      # Postavlja ime korisnika
git config --global user.email "email@domena.com"  # Postavlja email korisnika
git config --global core.editor "nano"          # Postavlja editor za Git
git config --list                               # Prikazuje sve Git postavke

2. Kreiranje i inicijalizacija repozitorija
------------------------------------------
git init                                        # Kreira novi lokalni Git repozitorij
git clone <url>                                 # Preuzima udaljeni repozitorij

3. Praćenje i dodavanje fajlova
-------------------------------
git status                                      # Prikazuje stanje repozitorija
git add <file>                                  # Dodaje fajl u staging area
git add .                                       # Dodaje sve promjene u staging area
git restore <file>                              # Vraća fajl iz posljednjeg commita
git restore --staged <file>                     # Uklanja fajl iz staging area

4. Commits
-----------
git commit -m "Poruka commita"                 # Snima promjene iz staging area
git commit -a -m "Poruka"                      # Commit svih praćenih fajlova

5. Pregled istorije
-------------------
git log                                         # Prikazuje istoriju commit-a
git log --oneline                               # Kratki pregled commit-a
git log --graph --oneline --all                 # Grafički prikaz svih grana i commit-a

6. Rad sa granama
-----------------
git branch                                      # Prikazuje lokalne grane
git branch <ime-grane>                          # Kreira novu granu
git checkout <ime-grane>                        # Prebacuje se na drugu granu
git checkout -b <ime-grane>                     # Kreira novu granu i prebacuje se na nju
git merge <grana>                               # Spaja drugu granu u trenutnu
git branch -d <grana>                           # Briše lokalnu granu

7. Rad sa udaljenim repozitorijem
---------------------------------
git remote -v                                   # Prikazuje udaljene repozitorije
git remote add origin <url>                     # Dodaje novi udaljeni repozitorij
git push -u origin <grana>                      # Šalje promjene na udaljeni repozitorij, i uspostavlja vezu izmedu dvije grane
git push                                        # Šalje lokalne promjene
git pull                                        # Povlači promjene sa udaljenog repozitorija

8. Undo i vraćanje promjena
---------------------------
git reset <file>                                # Uklanja fajl iz staging area
git reset --hard                                # Vraća cijeli repozitorij na posljednji commit
git revert <commit>                             # Kreira commit koji poništava određeni commit

9. Ostalo korisno
-----------------
git diff                                        # Prikazuje razlike između radnog foldera i staging area
git diff --staged                               # Prikazuje razlike između staging area i poslednjeg commita
git stash                                       # Privremeno čuva lokalne izmjene
git stash pop                                   # Vraća prethodno sačuvane izmjene
